# Real-time search

## Parent

Source: PRD.md (CopyVault MVP)

## What to build

A search bar at the top of the Home screen filters the visible snippet list in real time as the user types, matching against both title and content. No search button required.

## Acceptance criteria

- [x] Search bar is visible at the top of the Home screen
- [x] List filters as the user types, with no submit/search button
- [x] Matches are found in both title and content
- [x] Clearing the search bar restores the full list (newest first)
- [x] Filtering remains fast/smooth as the list grows
